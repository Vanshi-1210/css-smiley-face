# 😊 CSS Smiley Face

A fun and beginner-friendly CSS project that recreates a classic smiley face using only **HTML** and **CSS**. This project is great for practicing CSS positioning, shapes, borders, and basic styling techniques without using images or JavaScript.

## ✨ Features

- 😊 Pure HTML & CSS
- 🎨 Circular face design
- 👀 Symmetrical eyes
- 😊 Curved smiling mouth
- 🚫 No JavaScript required

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 What I Practiced

- CSS Positioning (`relative` & `absolute`)
- Border Radius
- Flexbox Alignment
- CSS Shapes
- Basic Layout Design
- Color Styling

## 🎯 Learning Outcome

This project helped me understand how simple CSS properties can be combined to create creative UI elements and improve my frontend fundamentals.
